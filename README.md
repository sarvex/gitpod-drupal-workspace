# Gitpod Drupal workspace
An opinionated Docker image you can use with Gitpod.io. Focused on Drupal 9.x development.


## Software

- composer 2.x
- ddev
- lando
- drush-launcher
- deployer
- robo
- drupalorg-cli
- scm_breeze
- husky
- rsync
- sass
- szeidler/composer-patches-cli
- ergebnis/composer-normalize
- roave/security-advisories
